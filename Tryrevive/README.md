# Tryrevive (Web Edition)

## 项目概述
**Tryrevive** 是一个纯净的心理学交互式网页项目（Web App / H5），旨在通过自适应测评（CAT）、情绪可视化网络图谱和流星雨呼吸减压动效，帮助用户恢复注意掌控权，与自我进行深度对话。

---

## 📂 项目结构
*   `README.md` - 本说明文档
*   `index.html` - 主页入口（状态火柴人与跳转白名单搜索框）
*   `meditation.html` - 底部圆形入口激活的 Galaxy 禅修空间（流星雨与泡泡自省文字）
*   `assessment.html` - 自适应测评与番茄倒计时界面
*   `dashboard.html` - 交互式情绪 SVG 拓扑关系图谱
*   （之后的所有前端静态网页、核心脚本都将存放于本目录下）

---

## 🎨 设计与技术规范
所有在本文件夹下编写的前端与系统代码必须严格遵循：
1.  **交互与缓动阻尼**：非线性的物理缓动，过渡效果应使用 `cubic-bezier(0.16, 1, 0.3, 1)` 等具有物理阻力感的缓动曲线。
2.  **排版宽度约束**：正文、卡片和问卷块宽度上限为 `max-width: 65ch`，防止视线发散。
3.  **色彩配比规范**：遵循 `60-30-10` 配色规律，中性背景降噪，辅助/信号高亮色占比小于 10%。
4.  **无 AI Clutter**：界面简洁纯净，不包含无意义的科技线条或炫技式动态背景。
